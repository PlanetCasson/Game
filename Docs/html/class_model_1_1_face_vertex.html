Detailed Description

abstract class for Face and Vertex class.

It is used to express the morphability between Faces and Verticies in a graph and its dual graph.

When the runtime class is Face:

The first dual Edge pointing from the center of the face towards an edge surrounding the face. The dual of this edge is an edge facing CCW direction bordering this face.

When the runtime class is Face:

The first dual Edge pointing from the center of the face towards an edge surrounding the face. The dual of this edge is an edge facing CCW direction bordering this face. In other words, this stores the first edge pointing out from the vertex of the dual graph that corresponds to this face.

When the runtime class is Vertex:

The first Edge pointing out from this vertex.

Note, this will throw an ArgumentException when the value set for EdgeListHead is not a valid value
